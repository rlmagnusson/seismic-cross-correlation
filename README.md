# Parallel cross-correlation for seismic phases

This Python module for calculating differential travel times for seismic phases using cross correlation. 
It is an attempt at combining several scripts into a portable module to simplify workflow. The calculation method for cross correlation is 
a version of the method used in [GISMO](https://github.com/geoscience-community-codes/GISMO) ported to numpy. 


This package is in very early stages, but some functionality is explained in example.ipynb.


Special thanks to Tom Winder for sharing his cross correlation scripts that this module is built on.
